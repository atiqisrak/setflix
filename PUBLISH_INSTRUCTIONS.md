# Publishing Setflix to GitHub

## Steps to Publish

### 1. Create Repository on GitHub

Visit: https://github.com/new

- **Repository name**: `setflix`
- **Description**: `A modern TV streaming platform built with Next.js`
- **Visibility**: Choose Public or Private
- **DO NOT** check "Add a README file" (we already have one)
- **DO NOT** check "Add .gitignore" (we already have one)
- **DO NOT** choose a license (or choose one if you want)

Click **"Create repository"**

### 2. Push Your Code

After creating the repository, run:

```bash
git push -u origin main
```

If you need to authenticate, GitHub will prompt you for credentials or use a personal access token.

### 3. Verify

Visit: https://github.com/atiqisrak/setflix
